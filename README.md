# 协同数据看板 (Light Data Board)

> More collaboration, better focus.

一款纯前端、零配置的极简数据透视与可视化看板工具。专为需要快速进行数据洞察的专业人士（如建筑师、工程师、项目管理者）设计。

旨在用轻量化的信息化技术替代繁琐的 Excel 数据透视操作，让团队成员脱离软件操作的泥沼，将精力重新聚焦在业务与设计本身。

## ✨ 核心特性

- **🚀 零安装，开箱即用**：纯前端单 HTML 文件架构。无需部署后端服务器，无需安装任何依赖（Node.js/Python），双击 `index.html` 即可在浏览器中直接运行。
- **🔒 数据绝对安全**：所有的数据解析、透视和图表渲染均在本地浏览器内存中完成，数据绝不上传云端，满足严格的企业保密需求。
- **📊 自动化数据透视**：支持自动识别 Excel/CSV 表头，一键完成 Group By（分组）和 Sum（求和）操作。
- **⚡️ 衍生指标计算**：内置轻量级特征计算引擎，支持对基础数据进行四则运算（如：成交价 ÷ 建筑面积 = 楼面价），即时生成衍生指标。
- **🔗 视图双向联动**：图表图例（Legend）与底层数据表单深度联动，支持表头点击排序，行状态隐现完全同步。

## 🛠 使用方法

1. 下载本仓库，或者直接将 `index.html` 文件保存到本地。
2. 使用任何现代浏览器（Chrome, Edge, Safari）双击打开 `index.html`。
3. 将你的 `.xlsx`、`.xls` 或 `.csv` 数据表格拖入网页。
4. 在控制面板中配置所需的维度（X轴）与指标（Y轴），点击“更新图表”。

## 📦 技术栈与开源合规说明

本项目基于原生 JavaScript 构建，无需构建工具，直接依赖以下优秀的开源项目（通过 CDN 引入）：

- **[Apache ECharts](https://echarts.apache.org/)** (Apache-2.0 License) - 提供核心的图表渲染与丝滑的交互动画。
- **[SheetJS (js-xlsx)](https://sheetjs.com/)** (Apache-2.0 License) - 在浏览器端进行极其强大的电子表格解析。
- **[Tailwind CSS](https://tailwindcss.com/)** (MIT License) - 通过原子化 CSS 快速构建现代化的极简界面。
