# 协同数据看板 (Light Data Board) - V3.0

> More collaboration, better focus.

一款纯前端、极简的数据透视与可视化看板工具。
旨在用轻量化的信息化技术替代繁琐的 Excel 操作，提高设计效率和质量，让业务团队能更专注于核心事务。

## 💡 核心设计理念

在日常业务协同中，主流的数据可视化方案往往伴随着门槛与妥协：

- **对比商业 SaaS (如 Datawrapper, Tableau)**：商业软件通常要求将基础数据上传至第三方云服务器，存在数据隐私泄露隐患，且操作链路繁琐。本项目采用纯前端本地读取，Excel 数据完全在浏览器内存中解析与透视，**“数据不落盘，绝不上云”**，确保敏感业务指标绝对安全。
- **对比重型开源框架 (如 Streamlit, Dash)**：数据科学框架通常依赖复杂的后端环境（如 Python 及依赖包），对非 IT 背景的业务团队极不友好。本项目实现**零环境依赖**，用户只需解压双击 HTML 文件即可在任意电脑断网离线运行，即拿即用。

| 核心维度     | Light Data Board (本项目)   | 商业 SaaS (如 Datawrapper) | 重型开源框架 (如 Streamlit) |
|:-------- |:------------------------ |:----------------------- |:-------------------- |
| **数据隐私** | **绝对安全 (本地浏览器内存计算)**     | 存在隐患 (需上传至第三方云端)        | 安全 (需自行部署内网服务器)      |
| **运行环境** | **零依赖 (双击 HTML 离线直接运行)** | 依赖外网 (必须在联网浏览器中运行)      | 强依赖 (需安装配置后端运行环境)    |
| **使用门槛** | **极低 (面向业务人员，拖拽即用)**     | 中等 (需注册账号与固定发布工作流)      | 高 (需掌握编程语言，面向开发者)    |

## ✨ V3.0 全新特性

V3.0 版本对底层架构进行了彻底重构，采用参数化与高度解耦的设计原则，将状态管理、计算引擎、图表策略与视图渲染严格分离。

- **🧩 Vue 3 模块化架构**：业务逻辑（透视算法）与视图组件（UI）深度解耦，支持低成本横向扩展新图表。
- **📊 自动化透视与多图表**：自动识别表头完成聚合计算。支持堆积柱状图、簇状柱型图与折线图（支持平滑曲线）的无缝切换。
- **⚡️ 衍生指标计算**：内置特征计算引擎，支持对基础数据进行四则运算，即时动态生成新指标。
- **🎨 行业级色彩策略**：内置莫兰迪 (Morandi)、商务蓝调 (Business) 与极简黑灰 (Monochrome) 多套专业配色方案，一键切换。
- **🖱️ 高性能交互表单**：底层数据表单重构，支持**鼠标平滑拖拽多选**、表头点击双向排序，并新增**悬浮合计栏**实时响应选中状态。

## 🛠 部署与开发向导

### 👨‍💼 面向普通用户 (开箱即用)

1. 在本仓库右侧 **Releases** 页面下载最新版本的 `index.html` 文件。
2. 双击打开离线单文件 `index.html` 即可在现代浏览器中使用。
3. 拖入 `.xlsx`, `.xls` 或 `.csv` 数据文件开始沉浸式分析。

### 👨‍💻 面向开发者 (二次开发)

本项目基于 Vite 构建，采用 Vue 3 + Tailwind CSS 现代化前端技术栈。

```bash
# 1. 克隆项目
git clone [https://github.com/nvbb/light-data-board.git]

# 2. 进入目录并安装依赖
cd light-data-board
npm install

# 3. 启动本地热更新开发服务
npm run dev

# 4. 构建生产环境离线单文件 (输出至 dist/index.html)
npm run build
```

## 📦 技术栈

-核心框架：Vue 3 (Composition API)

-样式引擎：Tailwind CSS (v4)

-数据渲染：Apache ECharts

-表格解析：SheetJS (js-xlsx)

-构建工具：Vite (配合 vite-plugin-singlefile 实现全量内联打包)
