# 协同数据看板 (Light Data Board)

> More collaboration, better focus.

一款纯前端、极简的数据透视与可视化看板工具。
旨在用轻量化的信息化技术替代繁琐的 Excel 操作，提高设计效率和质量，让设计师能更专注于设计本身。

## 💡 为什么选择 Light Data Board？

在日常业务协同中，主流的数据可视化方案往往伴随着门槛与妥协：

- **对比商业 SaaS (如 Datawrapper, Tableau)**：商业软件通常要求将基础数据上传至第三方云服务器，存在数据隐私泄露隐患，且操作链路繁琐。本项目采用纯前端本地读取，Excel 数据完全在浏览器内存中解析与透视，**“数据不落盘，绝不上云”**，确保敏感业务指标绝对安全。
- **对比重型开源框架 (如 Streamlit, Dash)**：数据科学框架通常依赖复杂的后端环境（如安装 Python 及各种依赖包），对非 IT 背景的业务团队极不友好。本项目实现**零环境依赖**，彻底摒弃后端，用户只需解压双击 `index.html` 即可在任意电脑断网离线运行，即拿即用。

| 核心维度 | Light Data Board (本项目) | 商业 SaaS (如 Datawrapper) | 重型开源框架 (如 Streamlit) |
| :--- | :--- | :--- | :--- |
| **数据隐私** | **绝对安全 (本地浏览器内存计算)** | 存在隐患 (需上传至第三方云端) | 安全 (需自行部署内网服务器) |
| **运行环境** | **零依赖 (双击 HTML 离线直接运行)** | 依赖外网 (必须在联网浏览器中运行) | 强依赖 (需安装配置后端运行环境) |
| **使用门槛** | **极低 (面向业务人员，拖拽即用)** | 中等 (需注册账号与固定发布工作流) | 高 (需掌握编程语言，面向开发者) |

## ✨ 核心特性

- **📊 自动化透视与多图表**：自动识别表头完成 Group By 与 Sum 操作。支持堆积柱状图、簇状柱型图与折线图的无缝切换。
- **⚡️ 衍生指标计算**：内置特征计算引擎，支持对基础数据进行四则运算，即时动态生成新指标。
- **🔗 视图双向联动**：图表图例（Legend）与底层数据表单深度联动，支持表头点击排序，数据行状态完全同步。
- **🧩 模块化架构**：基于 Vite 构建，核心逻辑与视图层严格解耦，易于二次开发与垂直业务定制。

## 🛠 使用方法

### 👨‍💼 面向普通用户 (开箱即用)
1. 在本仓库右侧 **Releases** 页面下载最新版本的 `.zip` 压缩包。
2. 解压后，双击打开 `dist` 目录下的 `index.html` 即可在浏览器中使用。
3. 拖入 `.xlsx`, `.xls` 或 `.csv` 数据文件开始分析。

### 👨‍💻 面向开发者 (二次开发)
本项目基于 Vite 构建，采用原生 JS 模块化架构搭建底层逻辑。

```bash
# 1. 克隆项目
git clone [https://github.com/nvbb/light-data-board.git]

# 2. 进入目录并安装依赖
cd light-data-board
npm install

# 3. 启动本地热更新开发服务
npm run dev

# 4. 构建生产环境离线单文件 (输出至 dist 目录)
npm run build

## 📦 技术栈

- 构建工具：**Vite** (配合 vite-plugin-singlefile 可实现单文件离线打包)

- 数据渲染：**Apache ECharts**

- 表格解析：**SheetJS (js-xlsx)**

- 样式排版：**Tailwind CSS**
