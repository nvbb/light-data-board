# 更新日志 (Changelog)

所有关于此项目的显著更改都将记录在此文件中。

## [2.0.0] - 2026-03-26

### 新增 (Added)

- **工程化重构**：全面引入 Vite 构建工具，实现了项目的模块化分离。

- **逻辑解耦**：将单文件脚本拆分为 `Store` (状态)、`Engine` (计算引擎)、`ChartFactory` (图表工厂) 和 `UIManager` (视图控制)。

- **图表形态切换**：支持在“堆积柱状图”、“簇状柱型图”和“折线统计图”之间实时切换。

- **数据衍生计算**：新增特征计算面板，支持用户通过 UI 界面对现有数据列进行加减乘除，动态生成新指标。

- **离线打包**：通过 `vite-plugin-singlefile` 支持将现代工程化代码打包为单 HTML 文件，保留双击即用的离线体验。

### 改进 (Changed)

- 优化了表格复选框的视觉交互，未选中行会自动降低透明度并沉降至表格底部。

- 增强了文件解析的容错性，自动清洗 Excel 表头中包含的回车与换行符。

## [1.0.0] - 初始版本

### 新增 (Added)

- 实现了纯前端读取 Excel/CSV 文件功能。

- 实现了基础的数据分类汇总（Group By）和求和（Sum）计算。

- 实现了基于 ECharts 的堆积柱状图与数据表格的双向联动（显示/隐藏）。
