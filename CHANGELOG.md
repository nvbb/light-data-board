# 更新日志 (Changelog)

所有关于此项目的显著更改都将记录在此文件中。

## [3.0.0] - 2026-03-26

### 🚀 Added (新增)

- **核心重构**：全面引入 Vue 3 (Composition API) 模块化架构，实现视图组件与底层透视计算引擎的严格解耦。
- **样式引擎**：接入 Tailwind CSS (v4) 替代手写原生 CSS，大幅提升界面现代化与响应式体验。
- **数据衍生引擎**：新增特征计算面板，支持对基础数值字段进行四则运算（加减乘除），即时生成并映射新指标（如计算楼面价等）。
- **行业级配色方案**：图表策略引擎新增主题切换功能，内置莫兰迪 (Morandi)、商务蓝调 (Business) 与极简黑灰 (Monochrome) 三套专业配色卡。
- **高阶交互表单**：
  - 底层数据表格新增**鼠标拖动多选**功能，支持批量平滑勾选/取消。
  - 增加底端悬浮**合计栏 (Total Row)**，实时计算并响应当前选中行的指标总计（内置浮点数精度处理）。
- **图表精细化控制**：折线图模式下新增“平滑曲线 (Smooth)”开关选项。

### 🔄 Changed (优化)

- **交互降噪**：将 X 轴（分类维度）由原先的下拉框形式改为平铺的“单选标签（Radio）”模式，缩短操作路径。
- **UI 响应流优化**：
  - 上传区域在载入数据后，自动收起并转换为右下角“全局悬浮重传按钮”，最大化释放图表阅读空间。
  - 彻底移除原有的“更新数据透视”手动触发按钮，实现全链路配置项的**点击即时热更新 (Reactive)**。
- **工程化构建**：使用 Vite 原生插件体系接管构建流程，支持将庞大的 Vue 工程极致打包为单一 `index.html` 脱机文件。

### 🛠 Fixed (修复)

- 修复了数据表单中因“选中项强制置顶”排序策略导致的“连环级联取消选中”交互 Bug。
- 修复了表格在拖拽多选时，容易触发浏览器原生拖拽行为而导致状态卡死（粘连）的问题。
- 修复了 ECharts 在 Vue 组件化挂载早期由于 DOM 时序问题导致的图表渲染空白 Bug（通过 `nextTick` 解决）。
- 修复了 Windows 环境下由于系统默认 GBK/ANSI 编码导致的构建失败问题（引入 `.editorconfig` 强校验）。

## [2.0.0] - 2026-03-26

### 新增 (Added)

- **工程化重构**：全面引入 Vite 构建工具，实现了项目的模块化分离。

- **逻辑解耦**：将单文件脚本拆分为 `Store` (状态)、`Engine` (计算引擎)、`ChartFactory` (图表工厂) 和 `UIManager` (视图控制)。

- **图表形态切换**：支持在“堆积柱状图”、“簇状柱型图”和“折线统计图”之间实时切换。

- **数据衍生计算**：新增特征计算面板，支持用户通过 UI 界面对现有数据列进行加减乘除，动态生成新指标。

- **离线打包**：通过 `vite-plugin-singlefile` 支持将现代工程化代码打包为单 HTML 文件，保留双击即用的离线体验。

### 改进 (Changed)

- 优化了表格复选框的视觉交互，未选中行会自动降低透明度并沉降至表格底部。

- 增强了文件解析的容错性，自动清洗 Excel 表头中包含的回车与换行符。

## [1.0.0] - 初始版本

### 新增 (Added)

- 实现了纯前端读取 Excel/CSV 文件功能。

- 实现了基础的数据分类汇总（Group By）和求和（Sum）计算。

- 实现了基于 ECharts 的堆积柱状图与数据表格的双向联动（显示/隐藏）。
